## Projeto

**Astro Runner Survival** é um jogo desenvolvido em Python utilizando a biblioteca Pygame. O jogador controla uma nave espacial e deve sobreviver a ondas crescentes de inimigos, enfrentando diferentes desafios e chefes ao longo da partida.

## Observações

Demais informações sobre instalação, execução e funcionamento do jogo podem ser encontradas no arquivo `README.md` localizado na raiz do projeto.

## Requisitos

O projeto foi desenvolvido e testado utilizando Python 3.12.

Recomenda-se utilizar Python 3.12 ou versões anteriores compatíveis. Versões mais recentes podem apresentar incompatibilidades com algumas bibliotecas ou funcionalidades utilizadas no projeto.

### Dependências

* Pygame

## Instalação

```bash
pip install -r requirements.txt
```

## Execução

```bash
python main.py
```

Ou, no Windows:

```txt
Play.bat
```
