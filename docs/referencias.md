# Referências e Créditos

Este projeto utiliza recursos de terceiros para trilha sonora e efeitos sonoros. Todos os direitos pertencem aos seus respectivos autores.

As músicas utilizadas no jogo são versões 8-bit inspiradas em obras de bandas conhecidas do rock e metal:
